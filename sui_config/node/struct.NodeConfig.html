Size of the broadcast channel used for notifying other systems of end of epoch.

In a sui-node binary, this is set to SupportedProtocolVersions::SYSTEM_DEFAULT in sui-node/src/main.rs. It is present in the config so that it can be changed by tests in order to test protocol upgrades.
