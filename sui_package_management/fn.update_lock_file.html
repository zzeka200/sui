Update the Move.lock file with automated address management info. Expects a wallet context, the publish or upgrade command, its response. The Move.lock principally file records the published address (i.e., package ID) of a package under an environment determined by the wallet context config. See the ManagedPackage type in the lock file for a complete spec.
